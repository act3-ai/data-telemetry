ACE Data Telemetry Server

The ACE Data Telemetry Server is responsible for tracking data usage and replica locations among all OCI registries that users utililize. The telemetry server only indexes the metadata from data bottles. The bulk/raw data is only stored in the original OCI registries. The telemetry server is populated primarily when users move data bottles with the ACE Data Tool.
