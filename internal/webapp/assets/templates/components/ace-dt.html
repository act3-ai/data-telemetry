ACE Data Tool

For more detailed information on ACE Data Tool and its features, reference the project git repository and usage guide.

In order to download/upload bottle metadata from/to Telemetry, you will need the ace-dt command-line tool installed on your system. You can find installation instructions here.

Using ACE Data Tool With Telemetry

NOTE: Authentication and/or a VPN connection might be required to access the Telemetry and/or registry servers.

To enable ace-dt to send and receive bottle metadata from this telemetry server, you will need to update your ace-dt configuration. You can either:

  • Set an environment variable: export ACE_DT_TELEMETRY_URL=
  • Update your ace-dt config file to include this telemetry server.

Once your ace-dt is properly configured, you may use the ace-dt pull and ace-dt push commands to download bottles and upload bottle info to Telemetry respectively.

For more information on configuring ace-dt click here or run ace-dt config --help
